Simple AL Slider is a great wordpress plugin and easy in setup. It will not only show your images but also represet your text info.

== Description ==

You can see demo at <a href="http://al-plugins.biz/simple-al-slider-box/">Simple AL Slider Live Demo</a>
Simple AL Slider is enough flexible and very easy to setup. You can add images and texts to any of your slide. It very helpful to show all benefits of your site for example.

### Main features of the slider

*   Capability to add images.
*   Capability to add texts.
*   Capability to make buttons.
*   Capability to attach css rules to text items and to slider.
*   Generate shortcodes to add to pages or posts or template file.
*   Capability to add links.
*   You can add a set of css rules to any text item.
*   You can add slider as widget.
*   Capability to setup right/left buttons in slider.
*   Two direction for moving slides.
*   Responsive design.


== Installation ==

1. Upload `simple_al_slider` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create new project
4. Upload images using upload button in "Slider" tab
5. Add texts using "create new text" button on "Texts" tab
3. Get codes from "Paste Code" tab and insert into page/post/template.
